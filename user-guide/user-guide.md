## Keyboard shortcuts

**Windows/Linux**
- `Ctrl + N` - New sync folder
- `Ctrl + R` - Refresh all syncs
- `Ctrl + P` - Pause all syncs
- `Ctrl + ,` - Open preferences

**macOS**
- `Cmd + N` - New sync folder
- `Cmd + R` - Refresh all syncs
- `Cmd + P` - Pause all syncs
- `Cmd + ,` - Open preferences
